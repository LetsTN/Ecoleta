## Antes de usar

Ir até <server/controllers/itemsController.ts> e <web/src/services/api.ts>, e alterar colocando o seu localhost.

## Para rodar

### server
```sh
npm run dev
```

### web
```sh
npm start
```

### mobile
```sh
npm start
```
